# Colors

Add a `theme.colors` object to provide colors for your project. By default these colors can be referenced by the `color`, `borderColor`, `backgroundColor`, `fill`, `stroke`, styles.

```js
// theme.js
export default {
  colors: {
    white: '#FFFFFF',
    black: '#000000',
    pink1: '#F571AB',
    pink2: '#C9427D',
    pink3: '#A43F6B',
    grey1: '#A5A5A5',
    grey2: '#707070',
    grey3: '#747474',
    grey4: '#393939',
    grey5: '#1C1C1C',
    red1: '#FF5E7C',
    red2: '#FF002F',
    red3: '#B60323',
    red4: '#84041C',
    teal1: '#0FB4B6',
    yellow1: '#F4D360',
    green1: '#8CBE72'
  }
};
```
