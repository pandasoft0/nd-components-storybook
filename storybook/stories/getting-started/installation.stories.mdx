## Installation

```bash
  npm i @district-ui
```

```bash
  yarn add @district-ui
```

## Set up Provider

```jsx
import * as React from 'react';
// 1. import `ThemeProvider` component
import { ThemeProvider } from '@district-ui/provider';

function App({ Component }) {
  // 2. Use at the root of your app
  return (
    <ThemeProvider>
      <>...</>
    </ThemeProvider>
  );
}
```
